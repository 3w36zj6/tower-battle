## Requirement
```
pip install -r requirements.txt
```

## Usage
It is recommended that you use [this plugin](https://obsproject.com/forum/resources/obs-virtualcam.949/).

```
python main_window.py <camera_id>
```

|key|description|
|:-:|:-:|
|up|Move up.|
|down|Move down.|
|left|Move left.|
|right|Move right.|
|a|Rotate counterclockwise.|
|d|Rotate clockwise.|
|space|Drop.|
|esc|Reset the game.|
